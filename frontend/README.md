# 🍽️ Recipe Web App

A full-stack recipe web application built with React, TypeScript, Node.js, and SQLite.
Browse through a variety of recipes and save your favorites for easy access.

## ✨ Features

- 🍪 Login/Registration uses express-session with cookie-based authentication and connect-sqlite3 as session store for persistent storage.
- 🔍 Browse through a variety of recipes
- ❤️ Save recipes to your favorites

## 🛠️ Tech Stack

| Layer    | Technologies                 |
| -------- | ---------------------------- |
| Frontend | React, TypeScript, HTML, CSS |
| Backend  | Node.js, Express             |
| Database | SQL                          |
